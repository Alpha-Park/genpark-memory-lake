# genpark-memory-lake

Local, private, and portable lifelong memory management for AI agents.

Stop paying for closed SaaS apps. This is a terminal-native, open-source alternative to **MemoryLake**.

Why trust a third-party cloud with your private thoughts? This skill creates a local 'data lake' of your context (using raw markdown and simple JSON embeddings), letting your GenPark agent retrieve a lifetime of memories securely and instantly.

## Usage
```bash
openclaw use genpark-memory-lake
```
